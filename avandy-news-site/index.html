Functionality
  • multi-user application with personalized interface settings and news filtering
  • search for news by one word or by several keywords for a period without filtering titles
  • search for all news using filtering
  • ability to add/remove/deactivate news sources (more than 20 RSS sources by default)
  • analysis of the frequency of words in news headlines - "Top 10"
  • display titles by double-clicking on a word from the "Top 10" for the period
  • the ability to exclude words from the "Top 10" and return them back
  • setting feelings from reading the news (positive/negative) and its significance (weight)
  • automatic setting of sensations according to predetermined rules (adding / deleting / deactivating rules for determining positive / negative)
  • built-in SQLite
  • Save titles to favorites
  • manually adding news to the archive
  • notification of pre-added events
  • quick jump to frequently used sites
  • export results to Excel and CSV
  • double-clicking on the heading opens the web page with the news
  • quick switching between users (you can have multiple users with different settings and filters)
  • to synchronize between computers, you can change the path to the database file in the application settings
  • saving interface state
  • various interface themes
  • set window transparency
  • program log
  • reader mode

Description

The "picture of the day" is analyzed for the most frequently used words in headlines using the Jaro-Winkler similarity method.

Each user can set his own keywords for both searching and excluding news, which allows you to customize the overall emotional background of the news received

Offline operation of the application is possible thanks to the built-in SQLite
